# PCOS Disease Prediction using Machine Learning
This project focuses on predicting the risk of Polycystic Ovary Syndrome (PCOS) using healthcare and clinical data. The workflow includes data cleaning, preprocessing, exploratory data analysis (EDA), and machine learning-based prediction.

## Objective
The goal of this project is to analyze PCOS-related medical parameters and build predictive models that can help identify important risk factors associated with the condition.

## Dataset
Public healthcare dataset used for educational and research purposes.

Dataset Source:
https://www.kaggle.com/datasets/shreyasvedpathak/pcos-dataset

## Workflow
* Data cleaning and preprocessing
* BMI correction and feature standardization
* Handling missing values
* Exploratory Data Analysis (EDA)
* Machine learning model training
* Feature importance analysis

## Tools & Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook
  
## Ethical Considerations
* No personal patient identifiers were used
* Dataset was publicly available
* Project created only for educational and research purposes
